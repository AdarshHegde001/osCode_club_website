# OS Code Prototype

A modern and responsive club website built for OS Code, designed to showcase the club’s activities, members, events, and workshops while providing an easy event registration system for students.

Tech Stack

Frontend

* React
* Vite
* React Router DOM

Backend

* Node.js
* Express.js

# Installation & Setup

```bash
# Clone the repository
git clone https://github.com/AdarshHegde001/osCode_club_website

# Navigate into the project folder
cd osCode_club_website

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will run locally at:

```txt
http://localhost:5173
```
